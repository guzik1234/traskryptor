# Instalacja pakietów językowych dla Argos Translate

## Aby tłumaczenie działało offline, zainstaluj pakiety językowe:

```bash
# Zainstaluj argostranslate
pip install argostranslate

# W Pythonie zainstaluj pakiety językowe:
python install_languages.py
```

## Pakiety językowe zostaną zainstalowane automatycznie przy pierwszym użyciu tłumaczenia.

Dostępne kierunki:
- Polski → Angielski
- Polski → Rosyjski

Po zainstalowaniu pakietów, tłumaczenie działa w pełni OFFLINE bez połączenia z internetem!
